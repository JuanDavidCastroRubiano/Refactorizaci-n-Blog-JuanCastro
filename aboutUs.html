¿Quiénes somos?

Somos un proyecto académico enfocado en el desarrollo web, donde se aplicaron conocimientos en HTML, CSS, Bootstrap y JavaScript para mejorar un blog deportivo.

Este sitio fue refactorizado con el objetivo de hacerlo más moderno, interactivo y adaptable a diferentes dispositivos, mejorando tanto su diseño como su funcionalidad.

Además, se buscó crear una experiencia más agradable para el usuario, incorporando estilos visuales, animaciones suaves y una mejor organización del contenido.

Este trabajo representa el aprendizaje adquirido durante el proceso de formación en desarrollo web.
